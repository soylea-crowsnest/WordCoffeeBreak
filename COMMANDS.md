# Word Guess — Voice Commands

| Command | When to Use | What It Does |
|---|---|---|
| **Thinking/Pause** | | |
| WAIT, HOLD ON, PAUSE, THINKING, NEED TIME, STAY, LET ME THINK, ONE MOMENT, SEC | During play | Enters thinking mode — no timeout pressure |
| READY, GO, OK, START, BEGIN, CONTINUE, YES, I'M READY, GUESS, BACK | In thinking mode | Resumes play and asks for your guess |
| **Guess Recall** | | |
| FIRST GUESS, GUESS 1, GUESS ONE | During play or thinking | Reads back that specific guess with letter results |
| SECOND GUESS, THIRD GUESS, etc. | During play or thinking | Same for any previous guess |
| LAST GUESS, PREVIOUS GUESS | During play or thinking | Reads back your most recent guess |
| RECAP, ALL GUESSES, MY GUESSES, SO FAR, SUMMARY | During play or thinking | Reads back all guesses with results |
| **Help** | | |
| HELP | During play | Explains how the game works |
| RULES | During play | Reads the full rules |
| HINT | During play | Reveals one unrevealed letter position |
| **Game Control** | | |
| REPEAT | Anytime | Repeats the last thing the game said |
| GIVE UP | During play | Reveals the answer, offers new game |
| QUIT, EXIT, STOP | Anytime | Ends the game |
| PLAY, AGAIN, YES, NEW | After win/loss | Starts a new game |
